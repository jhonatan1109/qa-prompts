## 🚀 ¿Cómo usar estos prompts?

1. Navega a la categoría que necesitas
2. Copia el prompt
3. Reemplaza los textos entre `[CORCHETES]` con tu información real
4. Pégalo en tu herramienta de IA favorita

### Ejemplo rápido

```
Antes:  [PEGA EL REQUISITO]
Después: El usuario puede iniciar sesión con email y contraseña.
         Si falla 3 veces, la cuenta se bloquea por 15 minutos.
```
